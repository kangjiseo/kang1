<h2>내 원격을 내 로컬 복제 (clone)</h2>
<hr>
♣git clone 원격저장주소 (뒤에 아무것도 없으면 현재 위치에) <br> <br>
♣ls -al 로 확인 <br> <br>
♣cd 해당저장소이름 <br> <br>
♣git branch -a (모든 브랜치명이 나온다) <br> <br>
♣git checkout 브랜치명 <br> <br>
♣ls -al (브랜치에 있던 파일이 나오는지 확인) <br> <br>
